# JavaWorld
欢迎来到学习java的世界

创建文件 HelloWorld.java
```
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
```
$ javac HelloWorld.java
$ java HelloWorld
Hello World
```
